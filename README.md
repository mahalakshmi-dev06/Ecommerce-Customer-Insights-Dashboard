# 📊 E-commerce Customer Insights Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** designed to analyze customer purchasing behavior using **RFM (Recency, Frequency, Monetary) Analysis**.

The dashboard helps identify different customer segments, understand customer value, and support data-driven business decisions.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Segment customers using RFM Analysis.
- Identify high-value and at-risk customers.
- Support business decision-making with interactive visualizations.

---

## 📈 Dashboard Features

- Customer Segmentation
- RFM Analysis
- Interactive Filters
- Monetary Analysis
- Frequency Analysis
- Treemap Visualization
- Scatter Plot
- Waterfall Chart
- Customer Summary Table

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
